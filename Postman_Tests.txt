POSTMAN TEST ENDPOINTS
======================

BASE URL: http://localhost:5000 (ou vérifiez le port affiché quand vous lancez `dotnet run`)

==========================================
TEST 1: MYSQL ONLY (Vehicles, Offers, Inspections)
==========================================

1. Seed MySQL Data
   POST http://localhost:5000/api/Test/seed-mysql
   Body: (vide)
   
2. Get All MySQL Data
   GET http://localhost:5000/api/Test/test-mysql
   
3. Test Search
   GET http://localhost:5000/api/Test/test-search

==========================================
TEST 2: MONGODB (Reviews, Audit Logs) - NÉCESSITE MONGODB INSTALLÉ
==========================================

Si MongoDB n'est pas installé, ces endpoints donneront des erreurs.

1. Seed MongoDB Data
   POST http://localhost:5000/api/Test/seed-mongodb
   Body: (vide)
   
2. Get All MongoDB Data
   GET http://localhost:5000/api/Test/test-mongodb

==========================================
CLEANUP
==========================================

DELETE http://localhost:5000/api/Test/cleanup

==========================================
INSTALLER MONGODB (Si nécessaire)
==========================================

Option 1 - MongoDB Community Edition:
https://www.mongodb.com/try/download/community

Option 2 - MongoDB avec Chocolatey:
choco install mongodb

Option 3 - Tester sans MongoDB:
Pour l'instant, testez seulement avec MySQL (Test 1)
